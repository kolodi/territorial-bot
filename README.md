## Environment variables

create a file names .env

put environemnt variables in there like so:

TOKEN - bot token
CLIENT_ID - application id
GUILD_ID - server id
STAGE - stages (like DEV or PROD)

## Register commands
Edit `register.js`
```npm run register```

## Command Permisisons
Edit `commands.permissions.js`
run
```npm run permissions```
